# Michigan MindMend Inc.

**Privacy-first, offline-capable AI tools for youth safety, family wellness, and responsible local automation.**

Michigan MindMend Inc. is a Michigan-built nonprofit technology effort founded by **Lyle Perrien II** in Owosso, Michigan. The mission is simple: build useful AI systems that help people without default cloud surveillance, hidden data harvesting, or platform dependency.

This GitHub organization is a portfolio of prototypes, research forks, demos, and infrastructure experiments across AI safety, local-first AI, privacy engineering, mental-health-adjacent support tools, and LLM systems work.

> These projects are not medical devices, crisis services, law-enforcement systems, or guaranteed production security products. They are engineering prototypes and open-source learning projects with clear safety boundaries.

---

## Featured Portfolio Projects

### [TrustLayer](https://github.com/MiMindMendinc/TrustLayer)
A lightweight LLM safety gateway for OpenAI-style chat-completion APIs. Demonstrates PII redaction, jailbreak/prompt-injection detection, topic blocking, streaming passthrough, and audit-log concepts.

**Shows:** LLM middleware, privacy filters, API routing, safety documentation.

### [MindMend Guardian](https://github.com/MiMindMendinc/mindmend-guardian)
A privacy-first youth safety prototype exploring local, family-controlled AI support, guardian-mode workflows, and offline-capable wellness architecture.

**Shows:** youth/family safety product thinking, local-first design, human-in-the-loop safety flows.

### [OpenClaw Empathy Anchor](https://github.com/MiMindMendinc/OpenClaw-Empathy-Anchor-MindMend-OpenClaw-)
An offline-first empathy and safety support layer for youth wellness, journaling, emotional check-ins, and safety-aware response generation.

**Shows:** Flask backend design, Node response-layer implementation, crisis-aware boundaries.

### [HumaniCare AI / MindMend Vault](https://github.com/MiMindMendinc/mindmend-vault-private)
A local-first starter scaffold for health, mental-health, and community-resilience tools with FastAPI endpoints, Docker Compose, tests, CI, safety notes, and audit logging.

**Shows:** FastAPI, Docker, pytest, GitHub Actions, safety/privacy documentation.

### [Grok-1 Inference Fork](https://github.com/MiMindMendinc/grok-1)
A public research fork of the open Grok-1 release focused on inference wiring, RoPE cleanup, benchmark entrypoints, and reproducible testing notes.

**Shows:** reading large model codebases, runtime testing, benchmark discipline, RoPE/inference work.

### [Dominus Ultra](https://github.com/MiMindMendinc/DominusUltra)
An educational Triton attention-kernel project with fused RoPE and GQA-style head mapping, framed around correctness-first GPU-kernel learning.

**Shows:** Triton, PyTorch, attention mechanics, correctness-first GPU experimentation.

### [SnapNote](https://github.com/MiMindMendinc/SnapNote)
A privacy-first photo-to-notes and summarization prototype designed around local/browser-side processing and simple family/caregiver workflows.

**Shows:** practical offline-first product UX, OCR/AI workflow design, privacy-first documentation.
